# MyTeams - Team Communication App

A professional Teams-like web application for team communication with real-time chat, file sharing, and team isolation.

## 🌟 Features

- **Team-Based Access**: Join with team codes, only same-team members can communicate
- **Real-Time Messaging**: Instant messaging with emoji support
- **File Attachments**: Share images, PDFs, documents with preview and download
- **Online/Offline Status**: Real-time status with "last seen" timestamps  
- **Persistent Chat History**: Messages saved across sessions and server restarts
- **Individual & Team Chats**: Private 1-on-1 chats and team channels
- **Mobile Responsive**: Works perfectly on phones, tablets, and desktops
- **Conversation Management**: Delete entire conversations
- **Unread Notifications**: Badge counts for unread messages
- **Auto-Delete Messages**: Customizable auto-cleanup (default 2 hours)

## 📱 How to Use

### 1. Join a Team
- Enter a **Team Code** (any text, e.g., "MYTEAM")
- Enter your **Name**
- Click "Join Team"

### 2. Start Chatting
- **Team Chat**: Click "Team Chat" (T icon) for group conversations
- **Private Chat**: Click any user's name for 1-on-1 messaging
- **Send Files**: Click 📎 to attach images, PDFs, documents
- **Emoji**: Click 😊 to add emojis to messages

### 3. File Sharing
- **Images**: Preview thumbnails, click for full-screen view
- **PDFs**: Click "Preview" to view without downloading
- **Documents**: Click "Download" to save to device
- **File Limit**: 10MB maximum per file

### 4. Team Management
- Only users with the same team code can see each other
- Offline users remain in sidebar with "last seen" time
- Delete conversations using ⋮ menu in chat header

## 🔧 Technical Features

### Built With
- **Backend**: Node.js, Express, Socket.io
- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Real-Time**: WebSocket connections
- **Storage**: File-based persistence (JSON)
- **UI/UX**: Microsoft Teams-inspired design

### Architecture
- **Real-time messaging** via Socket.io
- **File attachments** stored as Base64
- **Team isolation** via unique team codes
- **Session persistence** via localStorage
- **Responsive design** for all devices
